3. Spacing System (8dp Grid)

Alle Abstände basieren auf einem 8dp Grid-System für konsistente Layouts.

xs - 4px
sm - 8px
md - 16px
lg - 24px
xl - 32px
xxl - 48px

Anwendung

  • xs (4px): Minimale Abstände, Tag-Padding vertikal
  • sm (8px): Kleine Abstände zwischen verwandten Elementen, Filter-Chips-Spacing
  • md (16px): Standard-Spacing, Card-Padding, Section-Padding
  • lg (24px): Größere Abstände zwischen Sektionen
  • xl (32px): Große Abstände für visuelle Trennung
  • xxl (48px): Maximale Abstände für Desktop-Margins

8. Dos & Don'ts

✅ DOs - Best Practices

  • Dynamic Red IMMER für Cover-Ebene verwenden
  • 8dp Grid-System konsequent einhalten
  • Roboto Condensed Bold für alle Überschriften
  • Open Sans für Body Text
  • Mindestens 48dp Touch-Targets für mobile Geräte
  • Elevation zur visuellen Hierarchie nutzen
  • Barrierefreiheit: Min. Kontrast 4.5:1
  • Konsistente Border Radius: 8dp (Buttons, Inputs), 12dp (Cards), 16dp (Dialogs)
  • AppBar immer mit Elevation 4
  • Erfolgsfarbe Court Green für positive Aktionen

❌ DON'Ts - Zu vermeiden

  • Keine anderen Primärfarben als Dynamic Red
  • Keine gemischten Font-Families (nur Roboto Condensed + Open Sans)
  • Keine Touch-Targets unter 44dp
  • Keine inkonsistenten Spacings außerhalb des 8dp Grids
  • Keine flachen Buttons ohne ausreichenden Kontrast
  • Keine unleserlichen Text-Farb-Kombinationen
  • Keine Custom-Farben ohne Abstimmung
  • Keine unterschiedlichen Border Radius für gleiche Komponententypen
  • Nicht mehr als 2-3 Elevations gleichzeitig
  • Keine Shadows auf Paper-Hintergrund ohne Card

Button Component

// lib/core/widgets/rbs_components.dart

class RBSButton extends StatelessWidget {
  final String label;
  final VoidCallback? onPressed;
  final IconData? icon;
  final bool outlined;
  final bool textOnly;

  const RBSButton({
    super.key,
    required this.label,
    this.onPressed,
    this.icon,
    this.outlined = false,
    this.textOnly = false,
  });

  @override
  Widget build(BuildContext context) {
    if (textOnly) {
      return TextButton(
        onPressed: onPressed,
        style: TextButton.styleFrom(
          foregroundColor: RBSColors.dynamicRed,
          padding: const EdgeInsets.symmetric(
            horizontal: RBSSpacing.md,
            vertical: RBSSpacing.sm,
          ),
        ),
        child: Text(label),
      );
    }
    
    if (outlined) {
      return OutlinedButton.icon(
        onPressed: onPressed,
        icon: icon != null ? Icon(icon) : const SizedBox.shrink(),
        label: Text(label),
        style: OutlinedButton.styleFrom(
          foregroundColor: RBSColors.dynamicRed,
          side: const BorderSide(
            color: RBSColors.dynamicRed,
            width: 2,
          ),
          padding: const EdgeInsets.symmetric(
            horizontal: RBSSpacing.md,
            vertical: 12,
          ),
          shape: RoundedRectangleBorder(
            borderRadius: BorderRadius.circular(8),
          ),
        ),
      );
    }
    
    return ElevatedButton.icon(
      onPressed: onPressed,
      icon: icon != null ? Icon(icon) : const SizedBox.shrink(),
      label: Text(label),
      style: ElevatedButton.styleFrom(
        backgroundColor: RBSColors.dynamicRed,
        foregroundColor: RBSColors.white,
        elevation: 2,
        padding: const EdgeInsets.symmetric(
          horizontal: RBSSpacing.md,
          vertical: 12,
        ),
        shape: RoundedRectangleBorder(
          borderRadius: BorderRadius.circular(8),
        ),
      ),
    );
  }
}

Card Component

// lib/core/widgets/rbs_components.dart

class RBSCard extends StatelessWidget {
  final Widget child;
  final VoidCallback? onTap;

  const RBSCard({
    super.key,
    required this.child,
    this.onTap,
  });

  @override
  Widget build(BuildContext context) {
    return Card(
      color: RBSColors.white,
      elevation: 1,
      shape: RoundedRectangleBorder(
        borderRadius: BorderRadius.circular(12),
      ),
      margin: const EdgeInsets.only(bottom: RBSSpacing.md),
      child: InkWell(
        onTap: onTap,
        borderRadius: BorderRadius.circular(12),
        child: Padding(
          padding: const EdgeInsets.all(RBSSpacing.md),
          child: child,
        ),
      ),
    );
  }
}
